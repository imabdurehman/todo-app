# ✅ React Todo App

A responsive Todo App built with **React.js** that helps users manage daily tasks efficiently. The application supports complete CRUD operations, task status toggle, Local Storage for data persistence, and a clean, responsive user interface.

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as Completed/Pending
- 🔄 Toggle task status
- 🧹 Clear all tasks
- 💾 Local Storage support
- 📱 Responsive Design
- 🧩 Component-Based Architecture
- 🎨 CSS Modules

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- CSS Modules
- Local Storage
- Git & GitHub
- Vercel

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/imabdurehman/todo-app.git
```

Navigate to the project directory

```bash
cd todo-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```
